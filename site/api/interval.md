# interval()

Emits increasing integers `0, 1, 2, …` every `ms` milliseconds (Tier 2 source).

## Signature

```ts
function interval(ms: number): ProducerStore<number>
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `ms` | `number` | Tick interval. |

## Returns

`ProducerStore&lt;number&gt;`
